Result hiddbgApplyHdlsStateList ( const HiddbgHdlsStateList *  state)

Sets state for HiddbgHdlsStateList.

Only available with [7.0.0+]. The HiddbgHdlsState will be applied for each HdlsHandle. If a HdlsHandle is not found, code similar to hiddbgAttachHdlsVirtualDevice will run with the HiddbgHdlsDeviceInfo, then it will continue with applying state with the new device.
